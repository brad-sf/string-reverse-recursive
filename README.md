# StringReverseRecursive - JavaScript

## Installation
`npm install string-reverse-resursive`

## Usage
Node.JS:
```javascript
const {strReverse} = require('string-reverse-resursive');
console.log( strReverse("Hello") ); // olleH
```

ES6:
```javascript
import {strReverse} from 'string-reverse-resursive';
console.log( strReverse("Hello") ); // olleH
```
